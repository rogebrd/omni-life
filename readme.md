# Basic Setup
Backend runs on localhost:3001
Frontend runs on localhost:3000

To run:
in one terminal, run:
`cd backend/
npm run debug`

in another terminal, run:
` cd frontend/
npm run start`

Navigate to localhost:3000 to start the app!
